# Simpson Calculator

A fun calculator inspired by The Simpsons, with colorful buttons, a Springfield background, and Lisa Simpson on top.

**Live demo:** [https://paula-gi.github.io/calculadora/](https://paula-gi.github.io/calculadora/)

---

## Technologies Used

- React  
- Vite  
- HTML, CSS, and JavaScript

---

## Local Installation

To run the calculator on your computer, open a terminal and execute:

```bash
git clone https://github.com/paula-gi/calculadora.git
cd calculadora
npm install
npm run dev
